BUILD INSTRUCTIONS:
===================

1. Install i686-elf cross-compiler:
   sudo apt-get install gcc-i686-linux-gnu binutils-i686-linux-gnu
   
   OR build from source:
   wget https://ftp.gnu.org/gnu/gcc/gcc-11.1.0/gcc-11.1.0.tar.xz
   (See GCC documentation for cross-compilation setup)

2. Install build dependencies:
   sudo apt-get install make xorriso grub-pc qemu-system-x86

3. Build NyxOS:
   cd /workspaces/NYXOS
   make clean
   make              # Compiles kernel
   make iso          # Creates ISO image
   make run          # Launches in QEMU

KERNEL BOOT SEQUENCE:
=====================
1. GRUB loads multiboot kernel to 0x100000
2. entry.asm: setup stack, load IDT
3. kernel_main() starts:
   □ screen_init()       → VGA clear & init
   □ mem_init()          → Heap allocator startup
   □ idt_init()          → Setup IDT table
   □ cpu_init()          → CPUID detection
   □ irq_init()          → PIC remapping
   □ keyboard_init()     → PS/2 driver
   □ timer_init()        → PIT setup (100Hz)
   □ fs_init()           → Filesystem load
   □ sched_init()        → Scheduler setup
   □ enable_interrupts() → STI instruction
   □ shell_run()         → Command loop

KEY IMPLEMENTATION FILES:
=========================
- cpu.c:     CPUID leaf 0, 0x80000002-4, 1 for vendor/brand/family
- memory.c:  Block allocator with coalescing
- keyboard.c: Scancode table, interrupt buffer
- screen.c:  VGA mode driver with scrolling
- idt.c:     256-entry IDT setup
- irq.c:     PIC remapping (master to 0x20, slave to 0x28)
- fs.c:      Static VFS (/readme, /version, /ram)
- shell.c:   Input loop, command dispatch
